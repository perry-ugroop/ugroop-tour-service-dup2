
How to Run the Integration Test


1.) Open this same solution on 2 different instance of Visual Studio

2.) Rebuild the whole solution

3.) On the first IDE, modify the StormpathConfig.Initialize() properties then run the Ugroop.API.SQL project

4.) On the second IDE, modify the AccessToken on TourIntegrationTest then run the integration test

5.) You can see the test result on the second IDE.
